# Python语言程序与设计
## Movielens数据集爬虫
### 功能介绍
爬取Movielens数据集中电影在IMDB网站的评分，并存入CSV文件
### 开发环境
* 操作系统：windows10
* 开发语言： Python3
* 架构： Scrapy
* IDE： Pycharm

### 总结与体会
这次python的爬虫编写使自己收获不少。写程序不能一下要求太高，要着眼于全局，一步一步的慢慢实现，在加功能。
基本知识还是要掌握全面，使用框架爬取相关信息，注意到虽然框架提供了便捷，但是隐去了很多底层的实现，需要去多加了解程序的实现原理，多写代码使得自己获得较大的提高
